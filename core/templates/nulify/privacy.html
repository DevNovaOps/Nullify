Privacy Policy

How we handle and protect your data. Last updated: March 2026.

1. Information We Collect

When you use the Nullify platform, we may collect the following types of information:

  • Account Information: Username, email address, name, and role when you register.
  • Uploaded Files: Files you upload for PII detection and sanitization.
  • Usage Data: IP addresses, timestamps, and actions performed (stored in audit logs).

2. How We Use Your Data

  • To provide the PII detection and sanitization services
  • To maintain audit trails for compliance and accountability
  • To improve the platform and fix issues
  • To enforce security and prevent unauthorized access

3. Data Processing & Storage

All file processing happens locally on the server. No data is sent to external APIs or third-party cloud services. We use rule-based regex pattern matching for PII detection — no AI or machine learning models are involved.

Uploaded files and sanitized outputs are stored in the Django media folder on the server filesystem. You may delete your files at any time.

4. Data Security

We implement industry-standard security measures including:

  • Session-based authentication with secure cookies
  • Role-based access control (RBAC)
  • CSRF protection on all forms
  • Password hashing using Django's built-in security
  • Comprehensive audit logging of all actions

5. Data Retention

We retain your data only as long as necessary to provide the service. Uploaded files and sanitized outputs can be deleted by administrators. Audit logs are retained for compliance purposes.

6. Your Rights

You have the right to:

  • Access your personal data
  • Request correction of inaccurate data
  • Request deletion of your data
  • Export your data in a portable format

7. Contact

If you have questions about this privacy policy, please contact us at nullifyorg@gmail.com.
